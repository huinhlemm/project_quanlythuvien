# Hệ Thống Quản Lý Thư Viện

Hệ thống quản lý thư viện trực tuyến đa chức năng, được xây dựng bằng ASP.NET Core MVC.  
Dự án cung cấp giải pháp quản lý sách, mượn trả, phân quyền người dùng, tích hợp tìm kiếm sách bằng AI và tương tác thời gian thực.

## Tính năng chính

- Thiết kế hệ thống phân quyền RBAC (Admin – Librarian – User) đảm bảo bảo mật và kiểm soát truy cập
- Tìm kiếm sách thông minh bằng hình ảnh sử dụng AI Image Search
- Chat real-time hỗ trợ tương tác trực tiếp giữa người dùng và quản trị viên
- Tự động gửi email thông báo cho các tác vụ mượn/trả sách và reset mật khẩu
- Hỗ trợ import dữ liệu từ file Excel, giúp giảm thiểu thao tác nhập liệu thủ công
- Hoàn thiện toàn bộ quy trình với hơn 20 chức năng quản lý khác nhau

## Công nghệ sử dụng

- **Backend:** ASP.NET Core MVC
- **Realtime:** SignalR
- **Database:** SQL Server
- **ORM:** Entity Framework Core
- **AI Integration:** ONNX Runtime
- **Email Service:** MailKit
- **Khác:** Hỗ trợ xử lý file Excel

## Mục tiêu và Kết quả dự án

- Hoàn thiện một hệ thống quản lý thực tế với 20 chức năng cốt lõi
- Áp dụng thành công quy trình phát triển phần mềm end-to-end (từ thiết kế, phát triển đến triển khai)
- Thực hành tích hợp các module nâng cao như AI (ONNX) và Real-time (SignalR) vào hệ thống web truyền thống
